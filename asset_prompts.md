# SANDHYA — asset prompts (final)

**How to use:** each asset below has one grey code block. Copy the **whole block**, paste it
into Nano Banana, generate, then save the result with the **filename in the heading** into the
folder shown. Each block is self-contained — you never need to combine two blocks.

**Do these in order for a consistent look:** generate `icon.png` first. For every image after
that, stay in the **same Nano Banana chat** — the blocks already repeat the style, and staying
in one chat helps it match.

**Two background types (already written into each block):**
- *Cutout objects* (diya, bowl, blade, temple) come on a **flat magenta** screen — I delete that in code.
- *Full-bleed art* (icon, relics, scenes) keeps its own background.
